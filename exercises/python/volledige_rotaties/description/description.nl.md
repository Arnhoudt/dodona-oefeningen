Schrijf een programma dat telt hoevaak een minutenwijzer een volledige rotatie na delta minuten.

input:
m: startpunt van de minutenwijzer (in minuten)
d: delta minuten na de start (in minuten)

## Voorbeeld

```console?lang=python=>>>
input:
15
120
Output:
2
```

```console?lang=python=>>>
input:
55
150
Output:
2
```

```console?lang=python=>>>
input:
0
59
Output:
0
```
